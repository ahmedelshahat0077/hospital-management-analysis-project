# 🏥 Hospital Management SQL Analysis Project

## 📌 Project Overview

This project demonstrates a complete SQL-based data analysis workflow using a hospital management dataset. The goal is to transform raw healthcare data into meaningful business insights that support operational and financial decision-making.

The project simulates the responsibilities of a Data Analyst by performing:

- Data Quality Checks
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Business Analysis
- Performance Reporting
- SQL Joins and Aggregations

This project was built using Microsoft SQL Server.

---

## 🎯 Business Objectives

The analysis answers key business questions such as:

- How many patients and doctors are in the hospital?
- Which medical specialties have the highest number of doctors?
- What is the distribution of appointments by status?
- Which doctors generate the highest revenue?
- Which patients have unpaid bills?
- What are the hospital's total revenue and treatment costs?
- How does the patient journey flow from appointment to billing?

---

## 🗂️ Database Schema

The project contains five main tables:

| Table | Description |
|--------|-------------|
| Patients | Patient information |
| Doctors | Doctor information |
| Appointments | Patient appointments |
| Treatments | Treatment details |
| Billing | Billing and payment information |
